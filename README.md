# 📒 Address Book (Java Swing)

A simple Java GUI app to store and manage names and addresses.

## ✅ Features
- Add name & address  
- View saved entries  
- Clear all entries  

## 💾 Data
Entries are saved in `addressBook.txt`.

javac addressBook.java
